## Overview
This project is a user management web application system developed and tested for common security vulnerabilities. Security measures were applied to improve its protection.

## Work Done
- Tested for XSS vulnerability
- Tested for SQL Injection
- Identified weak password storage
- Implemented input validation (basic)
- Improved authentication (basic level)
